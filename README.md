# NFT Marketplace

## Структура проекта
Проект разделен на две части:
- `frontend/` - React приложение (клиентская часть)
- `backend/` - Express сервер (серверная часть)

## Запуск проекта

### Backend
```bash
cd backend
npm install
npm start
```
Сервер запустится на http://localhost:5000

### Frontend
```bash
cd frontend
npm install
npm run dev
```
Приложение будет доступно на http://localhost:8080

## Требования
- Node.js 18+
- MongoDB
